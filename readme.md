# 简约记账 App

## 项目说明
这是一个基于 Material Design 3 设计的简约记账应用，注重简洁的用户体验和实用的记账功能。

## 主要功能
- 快速记账
- 收支分类管理
- 月度收支统计
- 简洁的数据可视化

## 项目配置

### 开发环境
- Android Studio Version: Hedgehog | 2023.1.1
- Gradle Version: 8.2
- Kotlin Version: 1.9.0
- Java SDK Version: 17 (JDK)

### SDK 配置
- minSdk: 24
- targetSdk: 34
- compileSdk: 34
